# 🛠️ Calculator Universal Arme & Muniție - Familia Es Todo

Un calculator web interactiv și modern dezvoltat pentru gestionarea, planificarea și calculul resurselor necesare pentru fabricarea (crafting) de arme, componente, muniție și plicuri.

---

## 🌟 Caracteristici principale

* **Interfață dinamică cu Acordeon Inteligent:** Organizată pe categorii pliabile (**Arme**, **Componente Piese Armă**, **Muniție (Gloanțe)**, **Plicuri Cocaină**). Când deschizi o secțiune, celelalte se închid automat pentru o experiență curată și organizată.
* **Afișare Dinamică a Rezultatelor:** Panourile de rezultate din dreapta apar strict în momentul în care introduci date sau selectezi elemente în secțiunea corespunzătoare din stânga.
* **Calcul în timp real (Live):** Orice modificare a cantităților actualizează instantaneu cerințele și profitul estimat.
* **Agregare inteligentă:**
  * Calculează automat numărul de Blueprint-uri, Componente Piese Armă și piese specifice (țevi, corpuri, paturi de armă) în funcție de armele selectate.
  * Gestionează corect selecția directă a pieselor, calculând strict componentele brute necesare.
* **Management complet al muniției:** Include rețete detaliate pentru diverse tipuri de muniție (de la `.38 Magnum`, `.44 Marlin`, `7.62mm`, `.45 ACP`, `9mm` până la `.50 DAP`), calculând automat loturile, carcasele și materialele consumabile.
* **Secțiune dedicată Plicuri Cocaină:** Permite calcularea automată a necesarului de frunze, tăvi, sticle de apă, brichete, plicuri goale, greutate totală în KG și profitul estimat în funcție de resursa de pornire selectată.
* **Design Responsive & Dark Mode:** Un aspect vizual curat, modern, optimizat atât pentru ecrane mari (desktop), cât și pentru dispozitive mobile.

---

## 📁 Structura Fișierelor

Proiectul este conceput într-un singur fișier compact (Single-File Component / Standalone HTML), ideal pentru găzduire rapidă sau utilizare locală:

* `index.html` — Conține întreaga structură HTML, designul CSS personalizat și logica de calcul scrisă în JavaScript pur (Vanilla JS).
