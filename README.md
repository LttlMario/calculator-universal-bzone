# 🛠️ Calculator Universal Arme & Muniție - Familia Es Todo

Un calculator web interactiv și modern dezvoltat pentru gestionarea, planificarea și calculul resurselor necesare pentru fabricarea (*crafting*) de arme, componente și muniție.

## 🌟 Caracteristici principale

* **Interfață dinamică și intuitivă:** Organizată pe categorii pliabile (Arme, Componente Piese Armă, Muniție/Gloanțe) pentru o navigare rapidă.
* **Calcul în timp real (Live):** Orice modificare a cantităților din panoul de stânga actualizează instantaneu cerințele din panoul de rezultate din dreapta.
* **Agregare inteligentă:** 
  * Calculează automat numărul de **Blueprint-uri**, **Componente Piese Armă** și piese specifice (țevi, corpuri, paturi de armă) în funcție de armele selectate.
  * Gestionează corect selecția directă a pieselor de armă, scăzând redundanțele și calculând strict componentele brute necesare.
* **Management complet al muniției:** Include rețete detaliate pentru diverse tipuri de muniție (de la `.38 Magnum`, `.44 Marlin`, `.762mm`, `.45 ACP`, `9mm` până la `.50 DAP`), calculând automat loturile, carcasele și materialele consumabile (plumb, praf de pușcă, metale prețioase).
* **Design Responsive & Dark Mode:** Un aspect vizual curat, modern, optimizat atât pentru ecrane mari (desktop), cât și pentru dispozitive mobile.

---

## 📁 Structura Fișierelor

Proiectul este conceput într-un singur fișier compact (Single-File Component / Standalone HTML), ideal pentru găzduire rapidă sau utilizare locală:

* `index.html` — Conține întreaga structură HTML, designul CSS personalizat și logica de calcul scrisă în JavaScript pur (Vanilla JS).
